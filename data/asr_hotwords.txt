# 水货杀手 · 面试 ASR 热词库（计算机 / AI 领域）
#
# 用途：提高中英混合面试转写中「英文专有名词、缩写、易混技术术语」的识别准确率。
# 用法见 scripts/create_nls_vocabulary.py：
#   - 预览词表（粘贴到阿里云控制台）：python -m scripts.create_nls_vocabulary
#   - 导出权重 JSON：             python -m scripts.create_nls_vocabulary --json
# 在阿里云智能语音交互控制台「热词」用以上内容新建词表，得到 vocabulary_id 后
# 填入 .env 的 ALIYUN_NLS_VOCABULARY_ID，重启 gateway 生效。
#
# 格式：每行一个热词；# 开头为注释；空行忽略。权重由脚本按「英文缩写 > 英文词 > 中文术语」自动赋值。
# 只收录 ASR 容易错的术语；常见中文词（如「项目」「优化」）无需加，通用模型本就识别得好。
